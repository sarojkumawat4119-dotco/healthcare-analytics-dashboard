## Overview
Analyzed 10,000+ patient records using Python, SQL, Excel, and Power BI to identify disease trends, treatment outcomes, patient demographics, and hospital performance metrics. Built an interactive Streamlit dashboard for real-time data exploration.

## Technologies Used
Python, Pandas, NumPy, SQL, MySQL, Power BI, Advanced Excel, Streamlit, Plotly, Matplotlib, Seaborn

## Dataset
- Source: Kaggle Healthcare Dataset
- Size: 10,000+ patient records (54,966 rows after processing)

## Project Structure
- data/ - datasets and Excel analysis
- python/ - EDA notebook
- sql/ - 20 analytics queries
- dashboard/ - Power BI file and Streamlit app

## How to Run the Dashboard
cd dashboard
python -m streamlit run app.py
